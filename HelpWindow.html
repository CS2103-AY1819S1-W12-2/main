1. Introduction

Debt Tracker (DT) is for those who prefer to use a desktop app for tracking the user’s loans. DT has a mixed usage of Command Line Interface (CLI) and a Graphical User Interface (GUI). For more basic commands, one may use CLI, but DT has also simplified potentially longer commands through the use of GUI. DT can keep a record of your loans and debts as fast as other apps, if not faster!

2. Quick Start

  1. Ensure you have Java version 9 or later installed in your Computer.

  2. Download the latest debttracker.jar.

  3. Copy the file to the folder you want to use as the home folder for your Debt Tracker.

  4. Double-click the file to start the app. The GUI should appear in a few seconds.

  5. Type the command in the command box and press Enter to execute it. e.g. typing help and pressing Enter will open the help window.

Refer to Section 3, “Features” for details of each command.

3. Features

3.1. Adding a transaction : add

Adds a transaction to the database. Format: add n/NAME p/PHONE_NUMBER e/EMAIL a/ADDRESS d/DEBT dl/DEADLINE [t/TAG]…​

3.2. Analyse the your financial status : analytics

Analyse the your financial status and generate your financial status to view. It will either generate your financial status base on all the list, or generate to a certain date, base on the date you input. Format: analytics​ or Format: analytics​ DATE

3.3. Clearing all entries : clear

Clears all entries from the Debt Tracker. Format: clear

3.4. Currency conversion of transaction amounts: convert

Converts the transaction amounts from their respective foreign currencies to Singapore Dollars. Format: convert

3.5. Deleting a transaction : delete

Deletes the specified transaction from the address book. Format: delete INDEX

3.6. Editing a transaction : edit

Edits the fields of a particular transaction indentified by INDEX (must be a positive integer). Format: edit INDEX [n/NAME] [p/PHONE] [e/EMAIL] [a/ADDRESS] [t/TAG]

3.7. Exiting the program : exit

Exits the program. Format: clear

3.8. Filtering transactions: filter

Filter people out by whether they are a debtor or a creditor. Format: filter [t/TAG]…​

3.9. Locating transactions by transaction’s name : find

Finds transactions exchanged with any transaction whose name contains any of the given keywords. Format: find KEYWORD [MORE_KEYWORDS]

3.10. Viewing help : help

Invokes the user guide for handy reference when typing commands. Format: help

3.11. Hiding transactions with a private contact : hide

Makes transactions exchanged with people tagged as private invisible. Can be toggled. Format: hide n/NAME

3.12. Listing all recorded expenses : history

Lists all the commands that you have entered in reverse chronological order.

Lists all the expenses that you have recorded in the app, in reverse chronological order. Format: history

3.13. Adding interest rate to a debt : interest

Adds an interest amount to all transactions based on the interest scheme and interest rate you input. Format: interest INTEREST_SCHEME INTEREST_RATE

3.14. Lists all transactions which deadline is closest: nexttransaction

Finds and lists all transactions in the database which deadline is closest to current date. Format: nexttransaction

3.15. Redoing the previously undone command : redo

Reverses the most recent undo command. Format: redo

3.16. Removing a debt : remove

Remove away a number of debts from one transaction recorded in the DT using click-and-drag. The number of debts removed is up to the user’s selection.

3.17. Selecting a trasaction : select

Selects the transaction identified by the index number used in the displayed transaction list. Format: select INDEX

3.18. Sorting transactions : sort

Sort people by the amount of money loaned to or from them, the deadlines of the debts (be it whether you owe them or money or vice versa), or whether they are debtors or creditors. Format: sort [t/TAG]…​

3.19. Undoing previous command : undo

Restores the address book to the state before the previous undoable command was executed.
Format: undo

3.20. Updating a transaction : update

Edit an existing transaction’s details. Format: edit INDEX [n/NAME] [p/PHONE] [e/EMAIL] [a/ADDRESS] [t/TAG]…​

3.21. Saving the data

Debt Tracker data is saved in the hard disk automatically after any command that changes the data.

4. FAQ

Q: How do I transfer my data to another Computer?
A: Install the app in the other computer and overwrite the empty data file it creates with the file that contains the data of your previous Address Book folder.
